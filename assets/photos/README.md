# Photos

Drop your images into this folder using the exact filenames below. Wherever a
photo is missing, the site shows a tasteful labeled placeholder, so you can add
them one at a time and nothing ever looks broken.

| Filename        | Where it appears                | Best shape            |
|-----------------|---------------------------------|-----------------------|
| `hero.jpg`      | Full-screen top of the site     | Wide / landscape, large |
| `story-1.jpg`   | "How we met"                    | Portrait (tall)       |
| `story-2.jpg`   | "The proposal"                  | Portrait (tall)       |
| `banner.jpg`    | Full-width quote band           | Wide / landscape      |
| `gallery-1.jpg` ... `gallery-6.jpg` | Gallery grid | Square works best     |

Tips
- JPGs around 2000px on the long edge look crisp without being huge.
- Keep file sizes reasonable (under ~500 KB each) so the site loads fast.
- Want more or fewer gallery photos? Add/remove `gallery__item` blocks in
  `index.html` and matching files here.
